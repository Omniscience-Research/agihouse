Delight — AGI House display face (Phoenix concept)
==================================================

phoenix.html renders its all-caps display type with "Delight", the thin geometric
sans-serif from the AGI House deck (Salamahtype, via Befonts — see Befonts-License.txt).

Bundled, self-hosted, web-ready:
  Delight-VF.woff2      variable font, weight axis 100–900 (primary, ~34 KB)
  Delight-Light.woff2   static Light cut (fallback)
  Delight-Light.otf     static Light cut (opentype fallback)

The @font-face in phoenix.html declares font-weight:100 900 and the CSS uses
weight 300 for the big display lockups. No further setup needed.

(Jost on Google Fonts remains the final fallback if these files are ever missing.)
